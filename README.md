# da:gather
da:gather는 다문화 가정을 위한 친구 사귀기 플랫폼입니다.
다게더는 다문화 가정의 정보 취약, 외로움, 한국 생활 적응의 어려움 등의 문제점에서 출발하여,
다문화 가정 간의 커뮤니티 활성화를 위한 친구 사귀기 플랫폼을 제공합니다.
